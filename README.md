#  Real-Time Chat System

A full-stack real-time chat system built with:

- **Frontend**: Next.js (App Router)
- **Backend**: Express.js + Socket.io
- **Database**: MongoDB (via Mongoose)
- **WebSocket**: Real-time communication using Socket.io
